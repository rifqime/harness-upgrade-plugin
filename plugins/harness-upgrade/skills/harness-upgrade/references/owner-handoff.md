# Owner Answer Handoff

Use the repository contents, current docs, playbook baseline, and owner answers as the sources of truth for Pass 2.

Once the repo owner or main dev answers the Pass 1 questions:

1. Reconcile the owner answers with the repository contents and current docs.
2. Call out only true contradictions that affect the harness.
3. If the owner answers establish a policy or workflow, carry it forward explicitly into Pass 2.
4. If a point is still unknown, mark it as pending owner confirmation rather than inventing policy.

Do not let the handoff step become an excuse to fill gaps with guesses or to silently drop owner-approved nuance.
